Tuple
More than one element of same or different datatypes.
syntax1:
tuple_name=(ele1,ele2,ele3,.......)


Features:
1.Tuple is an immutable/unchangeable.
2.Tuple allows duplicates
3.Tuple is ordered-and index based.
4.No deletion/no insertion/no changes in the tuple.
Syntazx2:Tuple creation using Constructor
tuple_name=tuple((ele1,el2,el3,....))
